# Carpeta de Imágenes

## Logo del Sistema

Coloca tu archivo `logo.png` en esta carpeta.

**Ubicación del archivo:**
```
app/static/images/logo.png
```

**Requisitos recomendados:**
- Formato: PNG (con transparencia si es necesario)
- Tamaño: 200x200px o similar (se ajustará automáticamente)
- Fondo: Transparente o con el color de fondo del sistema

El logo se mostrará en:
- Sidebar del sistema
- Página de login (versión móvil y desktop)
